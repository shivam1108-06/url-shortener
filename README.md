# URL Shortener 🔗

A professional URL Shortener web application built using Flask, Python, Bootstrap 5, and JavaScript.

## Features

* Convert long URLs into short URLs
* Automatic URL redirection
* Copy short URL button
* URL history
* Click counter
* Loading animation
* Real-time date and time
* Responsive Bootstrap UI

## Technologies Used

* Python
* Flask
* HTML5
* CSS3
* Bootstrap 5
* JavaScript

## Installation

Clone the repository:

```bash
git clone https://github.com/shivam1108-06/url-shortener.git
```

Move to project folder:

```bash
cd url-shortener
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open in browser:

```text
http://127.0.0.1:5000
```

## How It Works

1. Enter a long URL.
2. Click "Shorten URL".
3. The application generates a unique short URL.
4. Open the short URL to get redirected to the original website.
5. View URL history and click counts.
